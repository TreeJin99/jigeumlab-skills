## 대학발 vs 지역발 핵심 차이

| 항목 | 대학발 (창업중심대학) | 지역발 (창업중심대학) |
|---|---|---|
| 주관 기관 | 창업중심대학 (대학) | 창업중심대학 (지역발 트랙) |
| **주력기술분야** | 27개 체크박스 | **없음** |
| **지원분야** | 없음 | **□ 제조 / □ 지식서비스** |
| **전문기술분야** | 없음 | **7개 체크박스** |
| 나머지 구조 | PSST 동일 | PSST 동일 |

> 지역발은 대학발(format-창업중심대학.md)과 구조가 거의 동일.
> 일반현황의 기술분야 체크박스만 다름. PSST 섹션 내용은 공통 패턴 참조.

## 문서 기본 형식

format-창업중심대학.md(대학발)와 동일:
- 표지 없음. 제목 박스(회색 배경) + □ 일반현황부터 시작
- 페이지 번호: `- N -` 형식, 하단 중앙
- 글머리: `ㅇ` (중항목), `-` (세부), `■` (소섹션)

### □ 창업아이템 개요(요약) ~ 3. 성장전략

대학발과 동일한 PSST 구조:
- □ 창업아이템 개요(요약)
- 1. 문제인식 (Problem): 1-1 배경및필요성, 1-2 목표시장현황분석
- 2. 실현가능성 (Solution): 2-1 준비현황, 2-2 실현및구체화고도화방안
- 3. 성장전략 (Scale-up): 3-1 비즈니스모델, 3-2 시장진입및사업화전략

## 지역발 고유 체크리스트

- [ ] 지원분야(제조/지식서비스) 택1 — **27개 주력기술분야 대신**
- [ ] 전문기술분야(7개 중 택1) 선택
- [ ] 총사업비 3분할: 정부지원/자기부담현금/자기부담현물
- [ ] 나머지 항목은 대학발 체크리스트와 동일 (format-창업중심대학.md 참조)

## 분량 가이드

대학발과 동일: **총 10~14쪽**

## 참고

지역발 공고에서 추가 고유 항목이 생길 수 있음 (지역 특화 항목, 지역 연계 내용 등).
당해 공고문 반드시 확인 후 대학발 포맷과 대조.
