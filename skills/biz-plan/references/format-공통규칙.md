## 모든 프로그램 공통 구조

### PSST 섹션 순서 (고정)
1. 일반현황
2. 아이템 개요(요약) 표
3. 문제인식 (Problem)
4. 실현가능성 (Solution)
5. 성장전략 (Scale-up)
6. 팀 구성 (Team)

### 문서 형식 공통
- 표지 없음 — 제목 박스 + 일반현황부터 시작
- 페이지 번호: `- N -` 형식, 하단 중앙
- 섹션 구분: `□` 기호 또는 숫자. 배너 박스
- 인쇄 기준: A4, 세로, 여백 표준
- **헤딩 레벨 절대 준수**: 공고문/포맷 파일에 명시된 헤딩 레벨(`#`, `##`, `###`)을 임의로 올리거나 내리지 않음

## 공통 문체 규칙

### 글머리 계층
```
ㅇ 중항목 (섹션 내 주요 포인트)
  - 세부 항목
  • 강조 항목 또는 글머리 목록
  ✓ 완료된 사실 또는 보유 역량
  → 결론/연결 문장
```

### 서술 금지 사항
- "혁신적인", "게임체인저", "압도적인" 등 추상 수식어 금지
- "~할 예정입니다" 식 서술체 금지 (개조식으로)
- LLM 자기평가형 표현 금지 ("잘 쓰여진", "훌륭한")
- 메타 설명 금지 ("다음과 같이 작성했습니다", "아래 내용을 참고하세요")
- 미확인 수치 단독 기재 금지 → `[확인 필요: {내용}]`으로 표기

### 금지 표현 → 권장 대체

| 금지 표현 | 권장 대안 |
|---|---|
| 혁신적인 솔루션 | 기존 대비 85% 개선된 솔루션 |
| 최초의 기술 | 국내 3개 기업만 보유한 기술 |
| 게임체인저 | 연 6,000만원 진입장벽을 제거하는 |
| 최첨단 AI | GPT-4 기반 정확도 95% AI |
| 빠른 성장 예상 | YoY +120% 성장 목표 |
| 큰 시장 규모 | 국내 시장 규모 3.2조원 (출처 명기) |

### 수치 표기 규칙
- **수치 밀도 기준**: 섹션당 최소 5개, 전체 30개 이상
- 모든 수치는 출처 명기: `(출처: 중소벤처기업부, 2023)` 또는 각주
- 퍼센트: `24.5%` (숫자+% 붙여 쓰기)
- 금액: 1억 미만은 `N백만원`, 1억 이상은 `N억원`
- 성장률: `YoY +333%` 또는 `전년 대비 3배`

### AI 생성 데이터 표기 구분

| 상황 | 표기 | 예시 |
|---|---|---|
| 클라이언트 미제공, AI 추정 | `(가상)` | "월 방문자 5,000명 (가상)" |
| 구조 설명용 예시 | `(예시)` | "경쟁사 가격 30만원 (예시)" |
| 근거 있는 추정치 | `(추정)` | "3년 후 매출 5억원 (추정)" |
| 출처 미확인 수치 | `[확인 필요: {내용}]` | "[확인 필요: 시장 성장률]" |

### 강조 패턴
- **굵게**: 핵심 수치, 결론 문장
- [대괄호]: 카테고리 분류 `[시장 기회]`, `[구체적 문제]`
- (소괄호): 보충 설명 `(출처: Statista, 2024)`
- < 꺾쇠 >: 도식/표 제목 `< 사업 추진일정 >`

## 공통 필수 항목

### 일반현황 (모든 프로그램)
- 창업아이템명 (2줄 이내, 핵심 가치 포함)
- 산출물 (협약기간 내 목표: 구체적 수량/종류)
- 팀 구성 현황 표 (순번/직위/담당업무/보유역량/구성상태)

### 아이템 개요(요약) 표
- 명칭 + 범주 (1행)
- 아이템 개요 (도식 + 핵심 키워드 + 특징)
- 문제인식 + 솔루션 + 성장전략 각 1셀
- 이미지 (실제 사진/화면 캡쳐 필수)

### Problem 섹션 필수 요소
- 시장 규모 수치 + 출처
- 기존 솔루션의 한계 (구조적 분석)
- 자사 사업 기회 연결 논리

### Solution 섹션 필수 요소
- As-Is vs To-Be 비교 (표 또는 도식)
- 현재 개발/검증 완료 근거 (수치, 사진, 인증서)
- 지식재산권 현황 (특허 출원/등록번호 포함)
- 사업 추진일정 (간트차트 형식)

### Scale-up 섹션 필수 요소
- TAM/SAM/SOM 시장 규모 (반드시 3단계 구분)
- 비즈니스 모델 (수익 채널 + 비중)
- 연도별 매출 목표 (최소 3개년)
- 경쟁사 비교표 (3개사 이상)

### Team 섹션 필수 요소
- 대표자 핵심 역량 (관련 경력 + 전문성 근거)
- 팀원 현황 또는 채용 계획

## 도식화 (나노바나나) 공통 요구사항

### 모든 섹션에 도식 1개 이상 필수

**아이템 개요 요약표**
- 서비스 플로우 (INPUT → PROCESS → OUTPUT)
- 사업 구조 도식 (파트너십, JV 등)

**Problem**
- 문제점-해결방안 2열 비교표 (색상 구분)
- 시장 성장 그래프 (bar/line)

**Solution**
- As-Is vs To-Be 비교 박스 2열
- 기술 아키텍처 (컴포넌트 블록 다이어그램)

**Scale-up**
- TAM/SAM/SOM 동심원 (3겹 원)
- 연도별 매출 타임라인 (4개 이정표)
- 경쟁사 포지셔닝 맵 (2×2 매트릭스) — 주요 프로그램

**Team**
- 업무파트너 로고 그리드 (청창사)

### 나노바나나 스타일 공통 제약
- 배경: #FFFFFF
- 폰트: Pretendard
- 색상 강조: 빨강(위기/기존대체제), 초록(기회/자사 강점), 파랑(중립)
- 카드형 금지: 두꺼운 컬러 바 사용 금지
- 화살표: 얇은 선 화살표 (굵은 화살표 금지)

## 자주 쓰는 표 템플릿

### 경쟁사 비교표
```
| 구분 | [경쟁사A명] | [경쟁사B명] | [자사명] |
|---|---|---|---|
| [핵심 항목1] | × | △ | ◎ |
| [핵심 항목2] | × | × | ○ |
| [처리 시간] | N시간 | N분 | N분 |
| [비용] | N만원 | N만원 | N만원 |
```

### 성과달성목표 (전년도↔목표 비교)
```
                    전년도('YY) 실적        협약기간 내 목표
정량적 성과
  매출              N만원                   N억원 (N배)
  채용              N명                     N명
  투자              N원                     N억원
  [기타 지표]       -                       N건/N%
정성적 성과         · [완료 사항]           · [목표 사항]
```

### TAM/SAM/SOM 표
```
| 구분 | 전체시장(TAM) | 유효시장(SAM) | 수익시장(SOM) |
| 정의 | | | |
| 규모 | N조원 | N억원 | N억원 |
| 근거 | | | |
```

## 공통 완료 체크리스트

모든 프로그램 공통. 프로그램별 추가 항목은 해당 `format-{프로그램}.md` 체크리스트 섹션 참조.

- [ ] 일반현황 표 모든 셀 채워짐 (빈 셀 없음)
- [ ] 아이템 개요(요약) 표 — 실제 이미지 삽입 위치 명시
- [ ] 개요(요약) 표 수치와 본문 각 섹션 수치 동기화 — OOO·[연도] 미완성 없음
- [ ] 모든 수치에 출처 또는 `[확인 필요]` 표기
- [ ] `[확인 필요]` 마커 0건 (제출 전 필수)
- [ ] TAM/SAM/SOM 3단계 이상 수치 명시 (LAM 포함 시 4단계)
- [ ] KPI·예산·추진일정 표 삽입 완료
- [ ] 팀 역량 → 사업 실현가능성 연결 문장 명시
- [ ] 빈 수식어("혁신적·게임체인저·패러다임") 제거
- [ ] 섹션별 자수·분량 제한 준수
- [ ] 도식 이미지 삽입 완료 (마커·드라이런·프롬프트 → 실제 이미지 링크 교체)
- [ ] 공고 양식 항목과 섹션 1:1 대응 확인
