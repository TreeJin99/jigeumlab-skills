### 1. 기술개발목표

**1-1. 기술개발의 필요성 및 목적**

ㅇ 시장 수요 및 기술 동향 (수치 근거)
ㅇ 기존 기술의 한계 (구조적 문제 명시)
ㅇ 기술개발의 필요성 → 결론 강조

**1-2. 과제 성능지표 달성 목표 (핵심 표)**

| 평가항목 | 설명 | 비중(%) | 세계최고수준 | 국내수준 | 1차년도 목표 | 2차년도 목표 |
- 정량적 측정 가능한 항목으로만 구성
- 세계/국내 수준 대비 달성 목표 수치 명시
- 비중 합계 = 100%

**1-3. 기대 효과**

ㅇ [기술적 확장 측면]
- 기술적 파급효과 (후속 연구, 원천기술 활용)
- 성능 개선 수치

ㅇ [사업적 확장 측면]
- 시장 진입 효과 (매출 목표, 고용 효과)
- 경제적 파급효과 수치

---

### 2. 연구개발방법

**2-1. 수행 내용 및 방법**

ㅇ 기술 개발 방법론
- 핵심 기술 구성요소별 개발 방법 (항목별 상세)
- 기술 개발 플로우차트 (선택)

**2-2. 연구 수행 체계 (산학연 고유)**

ㅇ 주관기업-위탁기관 역할 분담
| 구분 | 주관(기업) | 위탁(대학/연구기관) |
| 담당 내용 | | |
| 핵심 역할 | | |

ㅇ 연구팀 조직도 (주관 + 위탁 구분)

**2-3. 수행내용 × 기간 간트표**

| 추진내용 | 공통/주관/위탁 | MM | MM | ... |
- 공통/주관기업/위탁기관 행 분리 (산학연만)
- 단독기업(디딤돌)은 주관만

**2-4. 사업비 집행 계획 (연도별)**

| 비목 | 집행계획 | 정부지원사업비 | 자기부담(현금) | 자기부담(현물) | 합계 |
- 비목: 인건비/직접비(재료비/외주용역비/연구활동비 등)/간접비

## 디딤돌 vs 산학연 차이점

| 항목 | 디딤돌 (단독) | 산학연 Collabo |
|---|---|---|
| 연구 주체 | 기업 단독 | 기업(주관) + 대학/연구기관(위탁) |
| 연구팀 조직도 | 단순 | 이중 구조 (주관+위탁) |
| 수행내용 간트 | 단일 행 | 공통/주관/위탁 3행 분리 |
| 사업기간 | 보통 1~2년 | 예비연구(1년) + 사업화R&D(1년) 2단계 |
| 기술 수준 | TRL 3~6 | TRL 2~5 (예비연구 포함) |

## 연구개발계획서 고유 체크리스트

- [ ] **과제 성능지표 달성 목표표** (평가항목/비중/세계최고수준/국내수준/연도별목표) — 정량 지표만
- [ ] 비중 합계 100% 확인
- [ ] 세계최고수준/국내수준 수치 출처 명기
- [ ] 기대효과 [기술적 확장] + [사업적 확장] 분리 기재
- [ ] 연구개발방법 항목별 구체성 (재료/방법/기간 명시)
- [ ] 사업비 집행 계획 (비목별 현금/현물 분리)
- [ ] 산학연의 경우: 주관-위탁 역할 분담 표 + 이중 간트차트

## 분량 가이드

- 과제개요: 0.5쪽
- 자가진단: 0.5쪽
- 1. 기술개발목표: 3~5쪽 (성능지표 표가 핵심)
- 2. 연구개발방법: 4~6쪽 (간트차트 포함)
- **총 8~12쪽 추정** (공고마다 다름, 공고문 확인 필요)

## 도식화 포인트 [나노바나나 작업 대상]

- 과제 성능지표 달성 목표표: 세계/국내/1차/2차 비교 바 차트
- 기술 개발 플로우차트: 개발 단계 → 검증 → 산출물 흐름
- 연구팀 조직도 (산학연): 주관기업 + 위탁기관 2분할 박스
- 수행내용 간트표: 공통/주관/위탁 색상 구분 (산학연만)
- 기대효과: 기술적/사업적 2열 카드 구분
