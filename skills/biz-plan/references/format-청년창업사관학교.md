- (2) **개인적 동기**: 대표자 직접 경험 기반 (PMF 검증 포함 시 명시)

- `ㅇ {핵심 문장}` — 굵게
  - `- [시장 정의]` 수치 + 출처

**1-2-1. 현황 (2026 신규)**
