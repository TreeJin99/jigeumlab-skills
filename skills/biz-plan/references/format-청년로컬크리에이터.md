## 개인 로컬크리에이터 vs 청년 로컬크리에이터 핵심 차이

| 항목 | 개인 로컬크리에이터 | 청년 로컬크리에이터 |
|---|---|---|
| 대상 | 로컬 창업자 일반 | 청년 (나이 요건 공고 확인) |

| 섹션 구조 | <1.지역성> / <2.성장가능성> | 동일 구조 |

## 문서 기본 형식

- 표지 없음. 제목 박스 + □ 일반현황부터 시작
- 섹션 구분: < N. 항목 > (꺾쇠 배너) + 1) 소항목
- 글머리: `ㅇ` (중항목), `-` (세부)

### < 1. 지역성 >

**1) 지역이해도**

ㅇ 해당 지역(동네) 현황 및 특성
- 지역 인구/유동인구/상권 특성 (수치 근거)
- 지역의 강점 자원 (역사·문화·자연·산업 등)
- 지역 현안 또는 쇠퇴 요인

ㅇ 신청자의 지역 연계 경험
- 거주 기간 / 활동 이력 / 지역 네트워크

**2) 자원활용도**

ㅇ 활용하는 지역 자원 목록 및 방법
| 자원 유형 | 구체적 자원명 | 활용 방법 |
|---|---|---|
| 공간 | | |
| 인적 네트워크 | | |
| 문화·역사 | | |
| 원재료·생산 | | |

ㅇ 지역 파트너십 (협력 기관·상인·커뮤니티)

---

### < 2. 성장가능성 >

**1) 문제인식**

ㅇ 지역 문제 및 수요 정의
- 타깃 고객: (지역 거주자 / 방문객 / 특정 커뮤니티 등)
- 현재 해결되지 않는 문제 (수치 근거)
- 기존 대안의 한계

ㅇ 창업 동기 (개인적 경험 포함)

**2) 실현가능성**

ㅇ 핵심 서비스/제품 설명
- 현재 개발 완료 수준 (As-Is)
- 추가 개발 계획 (To-Be)

ㅇ 준비 현황
- 공간 확보 / 인허가 / 파트너십 / 시제품 등

ㅇ 추진 일정 (간트차트)

ㅇ 경쟁력 차별화
- 로컬 특화 요소 (타 지역 복제 불가한 요인)

**3) 지역가치창출가능성 (청년 고유 — 留繫活 체계)**

ㅇ 留(머무름) — 지역에 머무는 사람을 늘리는 효과
- 정주 인구 증가 / 이탈 방지 / 청년 정착 기여

ㅇ 繫(이음) — 지역 내외를 잇는 연결 효과
- 지역 주민↔방문객 연결 / 로컬 생산자↔소비자 연결
- 커뮤니티 형성 / 협력 네트워크

ㅇ 活(살림) — 지역 자원·경제를 살리는 효과
- 지역 매출 창출 / 일자리 / 유휴 공간·자원 활성화
- 지역 브랜드 가치 향상

## 청년 로컬크리에이터 고유 체크리스트

- [ ] **컨설팅 희망 분야** 체크박스 기재 (복수 가능)
- [ ] 로컬 유형 (5가지 중 택 1) 선택
- [ ] **留繫活** 3가지 지역가치창출가능성 각각 기재
- [ ] 지역이해도: 구체적 지역명 + 수치 근거
- [ ] 자원활용도 표: 자원 유형별 활용 방법 명시
- [ ] 지역 파트너십 현황

## 분량 가이드

- 일반현황: 0.5쪽
- 요약표: 0.5~1쪽
- 지역성: 2~3쪽
- 성장가능성: 3~4쪽
- **총 6~8쪽 추정** (공고문 확인 필요)

## 도식화 포인트 [나노바나나 작업 대상]

- 留繫活 3분류: 한자 아이콘 + 설명 3열 카드
- 지역 자원 활용도 표: 유형별 색상 구분
- 지역 현황 지도: 사업장 위치 + 주변 자원 표시
- 추진 일정 간트차트
