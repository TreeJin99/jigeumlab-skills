### 1. 문제인식 (Problem)

**1-1. 제품·서비스의 개발/개선 동기**
- 시장 규모 + CAGR 그래프
- 구조적 장벽 수치 (예: 95% 학원이 저작권 문제로 디지털화 불가)
- "Why Now?" 3가지 정책/환경 변화 카드
- → 결론 강조 박스: "특허 기반 해결 → 유일한 합법 솔루션"

**1-2. 창업아이템 목표시장(고객) 현황 분석**
- SOM/SAM/TAM 원형 도식 + 수치

ㅇ 고객군별 핵심 요구사항 및 딥테크 대응 기술 표 (딥창사 고유)
| 고객군 | 핵심 요구사항 | 객관적 근거 | **딥테크 대응 기술** |
(B2C / B2B학원 / B2B출판사 등 타겟별)

ㅇ 시장 특성 분석 (성장세/디지털전환/정책환경)

---

### 2. 실현가능성 (Solution)

**2-1. 창업아이템의 개발/개선 준비현황**

ㅇ 선행연구 개발현황 및 개선 계획

STEP 1 | 기반 인프라 (선행연구) 완료 현황
- 4개 완료도 카드: 완료율% + 항목명 + 근거 수치

STEP 2 | 딥테크 핵심 R&D (추가 개발)
- 3개 딥테크 엔진 카드
  - 엔진명 + 기반 기술 + 목표 수치 + TRL N~M + 현재 상태

- (선행 기술) 현재까지 개발 완료된 원천기술
- (축적 성과) 데이터·검증 현황 수치
- (현재 한계) 아직 부족한 부분 명확화
- (개선 계획) 딥테크 R&D 방향

ㅇ 최종 목표: AS-IS(현재) → TO-BE(딥테크 탑재 후) 도식

ㅇ 사업수행 및 실행력 관련 실적 (이미지 4개 카테고리)

## 딥테크창업사관학교 고유 체크리스트

- [ ] **권역 선택** (수도권/호남권/영남권 중 택1)
- [ ] **KPI 계량 택1** (고용실적/기술로얄티수익/투자유치)
- [ ] **KPI 비계량 택1** (인증획득/협력계약체결/지식재산권확보)
- [ ] **사업비 구성** 정부70%+입교자부담30%(현금10%+현물20%) 비율 확인
- [ ] 자가진단표(I.) 12개 항목 체크 + 서명
- [ ] **고객군별 딥테크 대응 기술 표** (청창사와 차별화되는 핵심 항목)
- [ ] **STEP1 선행연구 완료도% 카드** (4개, 각 근거 수치 포함)
- [ ] **STEP2 딥테크 핵심 R&D 카드** (3개, TRL 단계 명시)
- [ ] 지재권 현황 표 (출원번호 포함)
- [ ] 중복지원 검토 확인사항

## 분량 가이드

- 자가진단표: 1쪽
- 기본정보+일반현황: 2쪽
- 요약표: 1~2쪽
- Problem: 2~3쪽
- Solution: 3~4쪽 (딥테크 엔진 상세)
- Scale-up + Team: 3~4쪽 (공고문 확인 필요)
- **총 12~16쪽 추정** (청창사와 유사 분량)

## 도식화 포인트 [나노바나나 작업 대상]

- STEP1 완료도 카드: % 게이지 + 항목명 + 실적 수치 (4개 카드)
- STEP2 딥테크 엔진: 3개 카드 (TRL 바 + 목표 수치)
- 고객군별 딥테크 대응 기술 표 (딥창사 고유 — 타 프로그램 없음)
- 요약 진출목표시장: LAM/SOM/SAM/TAM 4분할 + 타겟 표
- AS-IS → TO-BE: 딥테크 탑재 전후 비교 도식
