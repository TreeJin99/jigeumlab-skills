## 로컬크리에이터 계열 3종 비교

| 항목 | 개인 로컬크리에이터 | 청년 로컬크리에이터 | **로컬기업육성사업** |
|---|---|---|---|
| 대상 | 개인 창업자 | 청년 창업자 | **기업 (법인/사업자)** |
| 분류 체계 | 5가지 유형 | 5가지 유형 | **4대 유형 + 7대 분야** |
| 사업비 구성 | 공고 확인 | 공고 확인 | **정부80% + 자부담20%** |
| 요약 고유 셀 | 없음 | 없음 | **비전(전략) + 사업화자금활용계획** |
| 지역성 구조 | 1)이해도 + 2)자원활용도 | 동일 | **1)지역이해도및자원활용도 (통합)** |
| 성장가능성 구조 | 문제인식+실현가능성 | +지역가치창출 | **+성장전략 (자금+시장)** |

### < 2. 성장가능성 >

**1) 문제인식**

ㅇ 지역 문제 및 시장 수요 정의
- 타깃 고객 (지역 내/외 고객군 구분)
- 현재 해결되지 않는 문제 (수치 근거)
- 기존 대안의 한계

ㅇ 창업/사업화 동기

**2) 실현가능성**

ㅇ 핵심 서비스/제품 설명

## 로컬기업육성사업 고유 체크리스트

- [ ] **4대 유형 (택 1)** 선택
- [ ] **로컬 7대 분야** 복수 선택 (공고문 항목명 확인 필수)
- [ ] **사업비 80/20 구성** 비율 확인 (다른 로컬 프로그램과 다름)
- [ ] 요약표에 **비전(전략)** 셀 포함
- [ ] 요약표에 **사업화자금활용계획** 셀 포함
- [ ] 지역이해도및자원활용도 **통합형** 작성 (개인/청년과 구조 다름)
- [ ] 성장전략 ①자금집행및조달계획 + ②시장진입및성과창출전략 분리 기재

## 분량 가이드

- 일반현황: 0.5~1쪽
- 요약표: 1쪽
- 지역성: 2~3쪽
- 성장가능성: 4~5쪽 (성장전략 포함)
- **총 8~10쪽 추정** (공고문 확인 필요)

## 도식화 포인트 [나노바나나 작업 대상]

- 4대 유형 + 7대 분야: 체크박스 배지 표시
- 지역 자원 활용도 표: 유형별 색상 구분 + 기여 효과 연결
- 비즈니스 모델: 수익원별 흐름 도식
- BEP 달성 그래프: 매출/지출 교차점 표시
- 단계별 시장 진입: 타임라인 화살표 도식
