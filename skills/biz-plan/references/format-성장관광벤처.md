## ⚠️ 구조 주의: PSST 아님

성장관광벤처는 일반 창업지원사업과 섹션 구조가 완전히 다름.
평가 핵심 축: **관광산업 연계성** + **검증된 사업모델** + **성장 가능성**

- 표지 없음. 제목 박스 + □ 일반현황부터 시작

### 1. 사업개요 및 특징 (2장 이내)

**1-1. 사업개발 동기 및 배경**

ㅇ (문제인식) 핵심 사회·시장 문제
- 뉴스/보고서 근거 + 3대 구조적 문제 도식 (아이콘 + 수치 카드)
- 구조: 언론이 말하는 현실 → 구조적 미스매치 → 시장 공백 = 기회

ㅇ (시장) 수치 기반 시장 규모 + 정책 환경
- TAM/SAM/SOM 또는 간략 수치

ㅇ (구조) 기존 솔루션의 한계
- 핵심 문제 항목 3가지 수치 포함

ㅇ (필요성) 자사 솔루션이 해결하는 이유
- → 결론 강조 박스

**1-2. 사업의 특장점**

ㅇ (사업 개요) 브랜드 슬로건 + 4단계 서비스 플로우
- 각 단계별 핵심 수치 포함 (누적 고객, MOU, 승인율, 매출 등)

ㅇ 포지셔닝 맵 (2×2 축)
- 축 설정: 온·오프라인 × 단일도시·전국 등 (관광 특성 축)
- 경쟁사 위치 표시, 자사 빈 포지션 강조

ㅇ 경쟁사 비교표 (자사 열 강조)
| 구분 | 경쟁사A | 경쟁사B | 경쟁사C | 자사 |

| 체험 지역 | | | | |

ㅇ 처별성·융합성·혁신성 3가지 강조 포인트

### 2. 시장 및 사업모델 (4장 이내)

**2-1. 시장현황 및 목표고객**

ㅇ 목표 시장 및 타겟 분석
- TAM/SAM/SOM/LAM 4분할 수치 + 각 정의

ㅇ 타겟 3분류 (B2C / B2B / B2G)
| 구분 | 1차 타겟(B2C) | 2차 타겟(B2B) | 3차 타겟(B2G) |
| 대상 | | | |
| 규모 | | | |
| 핵심 니즈 | | | |
| 객단가 | | | |
| 매출 비중 | 00% | 00% | 00% |

ㅇ 고객 니즈 근거 및 제공 솔루션 (타겟별 서술)

**2-2. 사업모델(Business Model) — 도식화**

ㅇ BM 도식
- 고객군 (B2C/B2B/B2G) → 서비스 허브 → 지역·기관 → 수익 채널

ㅇ 비즈니스 모델 산출 근거 표
| 수익 모델 | 단가 | 목표 수량 | 예상 매출 | 산출 근거 |

ㅇ 타겟별 비중 및 제공 서비스 표
| 구분 | 타겟 | 수요처 | 제공 서비스 | 비중 |

ㅇ LTV 구조 (1인당 단계별 매출 전환: 체험→유학→취업→정착)

**2-3. 서비스(상품)의 시장성**

ㅇ 사업 성과 현황 (이미지 + 수치 카드 6~8개)
- 언론보도/글로벌네트워크/사업검증·IP 등 3~4개 카테고리

ㅇ 시장 검증 및 고객 반응
- REVENUE / CUSTOMERS / GLOBAL 수치 3개 하이라이트
- 시장별 매출 비중 도식 (B2C/B2B/B2G)
- 전환율 / 고객 LTV / 파이프라인 지표

ㅇ 결론 강조 박스: "검증된 파이프라인 + 전환율 + LTV — 1회성이 아닌 단계별 매출 전환 구조"

**2-4. 잠재리스크 및 대응방안**

ㅇ 리스크 진단 및 대응 전략
- 리스크 3개 (HIGH/MID/MID 우선순위)
  - 각각: 리스크명 → 대응책 도식 (아이콘 + 방안)

ㅇ 리스크 대응 표
| 리스크 | 대응 방안 | 실행 시기 | 기대 효과 |

ㅇ 핵심 리스크별 구조적 대응 방안 상세 서술

## 성장관광벤처 고유 체크리스트

- [ ] **신청유형** (관광체험서비스/실감형관광콘텐츠/관광인프라/관광딥테크) 택1
- [ ] **투자단계** (Seed/Pre-A/Series A/그외) 표시
- [ ] **패스트트랙 해당여부** (관광벤처사업/관광액셀러레이팅) 체크
- [ ] **협약목표** (KPI/매출/투자유치/신규고용 4항목) 수치 기재

- [ ] 리스크 대응 도식 (HIGH/MID 우선순위 + 리스크별 구조적 방안)

## 도식화 포인트 [나노바나나 작업 대상]

- 요약 사업개요: 4단계 STEP 화살표 (각 수치 배지)
- 포지셔닝 맵: 2×2 축 (관광 특화 축 설정)
- 경쟁사 비교표: ×/○/△ 기호 (자사 열 강조색)
- BM 도식: 고객군 → 서비스허브 → 지역/기관 흐름
- LTV 타임라인: 체험→유학→취업→정착 단계별 금액
- 리스크 도식: 3개 리스크 카드 (우선순위 배지 + 대응책)
