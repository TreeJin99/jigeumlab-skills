# [C-5] 동심원 경쟁장벽 레이아웃 레퍼런스

> 발견 문서: A사 KOICA 사업계획서 (P5) — "4중 해자" 구조
> 용도: 중앙 브랜드를 보호하는 레이어별 진입장벽 시각화

---

## 레이아웃 구조

```
┌─────────────────────────────────────────────────────────┐
│                                                          │
│            ┌─────────────────────────┐                 │
│            │  ← 가장 바깥: IP 레이어  │                 │
│            │  ┌─────────────────┐    │                 │
│            │  │ ← TECH 레이어   │    │                 │
│            │  │  ┌───────────┐  │    │                 │
│            │  │  │ ← DATA    │  │    │  [외부 설명박스] │
│            │  │  │  ┌─────┐  │  │    │  IP: 설명 1줄   │
│            │  │  │  │브랜드│ │  │    │  TECH: 설명 1줄 │
│            │  │  │  └─────┘  │  │    │  DATA: 설명 1줄 │
│            │  │  └───────────┘  │    │  NET: 설명 1줄  │
│            │  └─────────────────┘    │                 │
│            └─────────────────────────┘                 │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## 구성 요소 상세

### 동심원 레이어 (안→밖)

| 레이어 | 내용 | 컬러 |
|---|---|---|
| 중앙 원 | 브랜드명 Bold | Primary 강조색 배경 + 흰색 텍스트 |
| 2번째 원 | 핵심 장벽명 (예: DATA) | 진한 계열 |
| 3번째 원 | 두 번째 장벽 (예: TECH) | 중간 계열 |
| 바깥 원 | 세 번째 장벽 (예: IP·NETWORK) | 연한 계열 |

**레이어 색상**: 안에서 밖으로 진→중→연 그라데이션. Primary 색상 계열 유지.

### 외부 설명 텍스트

원형 도식 우측 또는 사방에 각 레이어 설명 배치:
- 레이어명 SemiBold + 설명 1줄 Regular 소형
- 점선 또는 연결선으로 해당 레이어에 연결

---

## 나노바나나 프롬프트 핵심 지시

```
중앙에서 바깥으로 N겹 동심원:
- 가장 안쪽 원: "[브랜드명]" Bold 흰색, 강조색 배경 원형 도형
- 2번째 원: "[장벽명1]" 텍스트, [진한 강조색 계열] 원형 도형
- 3번째 원: "[장벽명2]" 텍스트, [중간 강조색 계열] 원형 도형
- 바깥 원: "[장벽명3]" 텍스트, [연한 강조색 계열] 원형 도형

우측 (또는 외부): 각 레이어별 설명
"[레이어명]" SemiBold + "[설명 1줄]" Regular 소형 회색 (반복)

동심원 = "원형 도형" 지시. 카드 컬러 바 금지 규칙과 별개.
[중요] 색상 코드는 디자인 지시사항. 이미지에 렌더링 금지.
```

## 컬러 적용 규칙

| 요소 | 컬러 | 비고 |
|---|---|---|
| 중앙 원 배경 | Primary 강조색 (진하게) | 도형 색상 (허용) |
| 2번째 원 배경 | Primary 80% | 도형 색상 (허용) |
| 3번째 원 배경 | Primary 50% | 도형 색상 (허용) |
| 바깥 원 배경 | Primary 20% 또는 #E5E7EB | 도형 색상 (허용) |
| 브랜드명 텍스트 | #FFFFFF Bold | — |
| 레이어명 텍스트 | #1A1A1A SemiBold | — |
| 설명 텍스트 | #6B7280 Regular | — |
