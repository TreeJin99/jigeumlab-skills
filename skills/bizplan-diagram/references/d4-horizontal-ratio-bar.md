# [D-4] 가로 분할 비율바 레이아웃 레퍼런스

> 발견 문서: B사 IR Deck (P14 자금사용계획), C사 IR Deck (P12 사업비구성)
> 용도: 자금구성·예산 항목별 비율을 가로 바로 한눈에 표현

---

## 레이아웃 구조

```
┌──────────────────────────────────────────────────────────┐
│                                                           │
│  ┌──────────────────────────────────────────────────┐   │
│  │ [항목A]  │  [항목B]  │  [항목C]  │ [항목D] │[E]│   │
│  │   35%    │    25%    │    20%    │   15%   │ 5%│   │
│  │ OOO만원  │  OOO만원  │  OOO만원  │OOO만원  │   │   │
│  └──────────────────────────────────────────────────┘   │
│       ↕              ↕             ↕                      │
│  [항목A 설명]  [항목B 설명]  [항목C 설명] ...           │
│   소형 텍스트   소형 텍스트   소형 텍스트                │
│                                                           │
└──────────────────────────────────────────────────────────┘
```

---

## 구성 요소 상세

### 가로 비율바

| 요소 | 형식 |
|---|---|
| 전체 바 | 전폭 가로 직사각형, 높이 60~80px |
| 구간 분할 | 항목별 비율에 따라 너비 분할 |
| 구간 경계 | 흰색 세로 선 (1~2px) |
| 구간 배경색 | 강조색~회색 그라데이션 분포 |

### 각 구간 내 텍스트

| 요소 | 형식 |
|---|---|
| 항목명 | SemiBold 중형 (흰색 또는 #1A1A1A) |
| % 수치 | Bold 대형 |
| 금액 | Regular 소형 |

### 하단 설명 연결 (선택)

바 아래 수직선 연결 + 항목별 상세 설명 Regular 소형.

---

## 나노바나나 프롬프트 핵심 지시

```
전폭 가로 비율바 1개를 N개 구간으로 분할:

[구간 나열 — 너비는 비율에 비례]
구간1: "[항목명A]" SemiBold + "[X%]" Bold + "[금액]" 소형 — [강조색] 배경
구간2: "[항목명B]" SemiBold + "[X%]" Bold + "[금액]" 소형 — [중간색] 배경
구간3: "[항목명C]" SemiBold + "[X%]" Bold + "[금액]" 소형 — [연한색] 배경
...

구간 경계: 흰색 세로 선 1px
바 전체 높이: 중간 높이 직사각형 (너무 얇지 않게)

[중요] 비율 수치(35%, 25%, 20%)와 색상 코드는 디자인 지시사항. 이미지에 렌더링 금지 문구 필수 포함:
"[중요] '35%', '25%' 등의 수치는 이미지에 표시할 텍스트입니다. 단, 레이아웃 너비 비율로 사용하는 지시사항 수치(비율바 너비 35% 등)는 이미지에 렌더링하지 마세요."
```

## 컬러 적용 규칙

| 구간 | 컬러 | 비고 |
|---|---|---|
| 1번째 구간 | Primary 강조색 (진하게) | 도형 배경 (허용) |
| 2번째 구간 | Primary 70% | 도형 배경 (허용) |
| 3번째 구간 | Primary 50% | 도형 배경 (허용) |
| 4번째 이후 | Primary 30%, 20%, #9CA3AF | 도형 배경 (허용) |
| 구간 내 텍스트 | 배경이 진하면 흰색, 연하면 #1A1A1A | 가독성 기준 |
| 구간 경계선 | #FFFFFF 1px | — |

## 파이차트 대신 비율바를 선택하는 이유

- 항목이 5개 이상이면 파이차트 가독성 급감
- 가로 바는 항목명+%+금액 3가지를 한 공간에 모두 표시 가능
- 배너 형태로 슬라이드 전폭 활용, 시각적 임팩트 큼
