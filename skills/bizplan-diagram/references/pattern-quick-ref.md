# 도식 코드별 프롬프트 구조 요약 (A-1 ~ E-2)

> 각 패턴의 레이아웃·이미지 텍스트·컬러 스펙. Phase 3 ASCII 드라이런 작성 시 참조.
> 상세 스펙이 있는 패턴(B-1, B-3, B-4, C-4, C-5, D-3, D-4, E-1, E-2, A-4)은 해당 references 파일 우선.

## A. 제안배경 / 문제인식

### [A-1] 넘버링 문제 카드
- **레이아웃**: 가로 나열 카드 3~4장
- **이미지 텍스트**: ①②③ 번호 + 키워드 제목 + 설명 + 핵심 수치 Bold 대형 (있으면)
- **컬러**: 번호 텍스트에만 강조색

### [A-2] 3중 벤다이어그램
- **레이아웃**: 삼각 배치 반투명 원 3개 겹침
- **이미지 텍스트**: 원①②③ 이름+수치, 교차점 브랜드명
- **컬러**: 교차점 텍스트에만 강조색

### [A-3] 풀배너 결론박스
- **레이아웃**: 슬라이드 하단 전폭 강조색 배경 배너
- **이미지 텍스트**: 핵심 결론 문장 1줄 (Bold 흰색)
- **컬러**: 배너 배경 강조색, 텍스트 흰색

### [A-4] 수렴 브라켓
- **레이아웃**: 좌측 항목 2~4개 세로 나열 → 우측 꺾임 브라켓 → 결론 텍스트+CTA박스
- **이미지 텍스트**: 각 항목 번호+제목+체크리스트 / 우측 결론 Bold + CTA 1줄
- **컬러**: 번호 태그박스 강조색, CTA박스 배경 강조색, 브라켓 선 강조색
- **상세 스펙**: `references/a4-convergence-bracket.md`

## B. 시장분석

### [B-1] TAM/SAM/SOM (→LAM 4단계 확장 가능)
- **레이아웃**: 좌우 분할 — 좌측 반원 아크 3겹(기본) 또는 4겹(LAM 포함 시) + 우측 N행 정보 패널
- **컬러**: TAM=네이비, SAM=스틸블루, SOM=연회색, LAM=연회색(내원) / 수치 텍스트 강조색
- **LAM 4단계 사용 시**: 아크를 4겹으로 지시. LAM행 우측에만 박스 강조 (흰색+drop shadow). 4단계 순서: TAM→SAM→SOM→LAM(중심)
- **나노바나나 주의**: 아크는 "반원 형태의 도형"으로 지시 — 카드 컬러 바 규칙과 별개
- **상세 스펙**: `references/b1-tam-sam-som.md`

### [B-2] 경쟁사 O/X 비교
- **레이아웃**: 표 (행=기능, 열=경쟁사+우리)
- **이미지 텍스트**: 기능명·O/X/△·결론 캡션
- **컬러**: O=초록 텍스트, X=빨강 텍스트, 우리 열 배경 강조색 (열 전체)

### [B-3] 포지셔닝 매트릭스 2×2
- **레이아웃**: X/Y축 교차 4사분면 + 경쟁사 원형 버블 + 자사 버블 강조
- **이미지 텍스트**: X축 양끝 레이블 2개, Y축 양끝 레이블 2개, 각 버블에 경쟁사명
- **컬러**: 자사 버블만 강조색, 경쟁사 버블 회색. 축선 #1A1A1A 얇은 선
- **나노바나나 주의**: 버블은 "원형 도형"으로 지시 — 카드 컬러 바 규칙과 별개
- **상세 스펙**: `references/b3-positioning-matrix.md`

### [B-4] 막대그래프 + 수치강조패널
- **레이아웃**: 좌측 연도별 막대그래프(3~5개) + 우측 핵심수치 2~3개 스택
- **이미지 텍스트**: 막대 위 수치 라벨, CAGR 화살표 라벨 / 우측 수치 Bold 대형 + 설명 소형
- **컬러**: 미래 연도 막대 강조색, 현재·과거 막대 회색·연색. 우측 수치 강조색
- **상세 스펙**: `references/b4-bar-chart-kpi-panel.md`

## C. 솔루션 / 차별성

### [C-1] AS-IS → TO-BE
- **레이아웃**: 좌우 2분할 또는 고객군별 카드 (AS-IS + ↓/→ + TO-BE)
- **이미지 텍스트**: AS-IS: ✗ 문제 3개 / TO-BE: ✓ 해결 3개
- **컬러**: ✗=빨강 텍스트, ✓=초록 텍스트. 화살표 강조색

### [C-2] 선순환 플로우
- **레이아웃**: 원형 순환 화살표, 중앙 브랜드명
- **이미지 텍스트**: STEP 01~04 제목·설명, 중앙 브랜드명
- **컬러**: 핵심 STEP 텍스트에만 강조색, 나머지 #1A1A1A

### [C-3] 파트너십 허브
- **레이아웃**: 중앙 원 + 4방향 연결선 + 파트너 박스
- **이미지 텍스트**: 중앙 브랜드명, 파트너 기관명·역할 (4개)
- **컬러**: 중앙 원 배경 강조색+흰색 텍스트, 파트너 박스 흰색

### [C-4] SWOT 2×2 사분면
- **레이아웃**: 2×2 사분면 (S·W·O·T) + 하단 전략카드 4개 (선택)
- **이미지 텍스트**: S/W/O/T 레이블 Bold 대형 + 각 셀 불릿 3~4개
- **컬러**: S·O 셀 배경 강조색(연하게), W·T 셀 배경 연회색. 레이블 강조색 Bold
- **상세 스펙**: `references/c4-swot-quadrant.md`

### [C-5] 동심원 경쟁장벽
- **레이아웃**: 중앙 브랜드명 + 바깥으로 레이어 2~4겹 (해자 구조)
- **이미지 텍스트**: 중앙 브랜드명, 각 레이어에 장벽명 (DATA·NETWORK·TECH·IP 등)
- **컬러**: 레이어 도형은 진→중→연 계열, 중앙 텍스트 강조색
- **나노바나나 주의**: 동심원은 "원형 도형"으로 지시 — 카드 컬러 바 규칙과 별개
- **상세 스펙**: `references/c5-concentric-moat.md`

## D. 성장전략 / 로드맵

### [D-1] STAGE 로드맵
- **레이아웃**: 좌→우 박스 3개 + 화살표
- **이미지 텍스트**: STAGE 이름·시기·핵심지표·주요과제
- **컬러**: 현재 STAGE 박스 강조색 배경, 나머지 흰색+회색 테두리

### [D-2] 사업전후 비교표
- **레이아웃**: 표 (행=지표, 열=현재/목표/비고)
- **이미지 텍스트**: 매출·수출·투자·고용 수치
- **컬러**: 목표 열 수치 텍스트 강조색

### [D-3] 마일스톤 지그재그 타임라인
- **레이아웃**: 가로 중앙선 + 홀수 마일스톤 위·짝수 아래 교번 배치
- **이미지 텍스트**: 각 마일스톤: 아이콘 + 날짜/분기 + 제목 Bold + 설명 소형
- **컬러**: 현재 마일스톤 원형 도트 강조색, 과거 회색, 미래 빈 원
- **상세 스펙**: `references/d3-zigzag-timeline.md`

### [D-4] 가로 분할 비율바
- **레이아웃**: 전폭 가로 바 1개를 N개 구간으로 분할 (자금구성/예산)
- **이미지 텍스트**: 각 구간: 항목명 + % Bold 대형 + 금액 소형
- **컬러**: 구간별 강조색~회색 그라데이션 분포. 구간 경계 흰색 선
- **나노바나나 주의**: 비율 수치(30%, 70%)는 이미지에 렌더링 금지 문구 별도 명시
- **상세 스펙**: `references/d4-horizontal-ratio-bar.md`

## E. 사업개요 / 비즈니스모델

### [E-1] 2컬럼 서비스 개요 인포그래픽
- **레이아웃**: 16:9, 헤더(전체 너비) + 좌컬럼(핵심기능) + 우컬럼(목업형/플로우형)
- **상세 스펙**: `references/e1-overview-infographic.md`

### [E-2] KPI 하이라이트 카드
- **레이아웃**: 5개 수치 카드 — 상단 2개(전폭 분할) + 하단 3개(가로 나열), 중앙 브랜드 원형
- **이미지 텍스트**: 각 카드: 카테고리명 소형 + 수치 Bold 최대형 + 설명 소형
- **컬러**: 수치 텍스트에만 강조색. 카드 배경 흰색, 구분선 회색
- **상세 스펙**: `references/e2-kpi-highlight-cards.md`
