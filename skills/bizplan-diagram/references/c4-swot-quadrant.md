# [C-4] SWOT 2×2 사분면 레이아웃 레퍼런스

> 발견 문서: B사 IR Deck (P8), A사 KOICA 사업계획서 (P7)
> 용도: 강점·약점·기회·위협을 2×2 사분면으로 시각화

---

## 구성 요소 상세

### 사분면 셀 구성

| 셀 | 레이블 | 배경 | 내용 |
|---|---|---|---|
| 좌상 (S 강점) | "S" Bold 대형 + "강점" 소형 | Primary 강조색 연하게 (10~20% 불투명도) | 불릿 3~4개 |
| 우상 (O 기회) | "O" Bold 대형 + "기회" 소형 | Primary 강조색 연하게 | 불릿 3~4개 |
| 좌하 (W 약점) | "W" Bold 대형 + "약점" 소형 | #F3F4F6 연회색 | 불릿 2~3개 |
| 우하 (T 위협) | "T" Bold 대형 + "위협" 소형 | #F3F4F6 연회색 | 불릿 2~3개 |

**중앙 교차점**: 회사명 또는 브랜드명 작게 배치 (선택)

### 하단 전략카드 (선택)

SWOT 사분면 아래에 SO·ST·WO·WT 전략 카드 4개 가로 배열. 카드당: 레이블 Bold + 전략 불릿 2개.

---

## 나노바나나 프롬프트 핵심 지시

```
2행 2열 사분면 표:
- 좌상 "S 강점" 셀: Primary 강조색 연한 배경 (약 15% 불투명도), "S" Bold 대형 강조색 + "강점" 소형 + 불릿 내용
- 우상 "O 기회" 셀: 동일 강조색 연한 배경
- 좌하 "W 약점" 셀: #F3F4F6 연회색 배경, "W" Bold 대형 + "약점" 소형 + 불릿 내용
- 우하 "T 위협" 셀: 동일 연회색 배경
- 셀 경계: 1px 회색 선

[중요] 배경색 값은 디자인 지시사항. 이미지에 렌더링 금지.
S/O 셀 배경 강조색 — 카드 컬러 바 규칙과 별개 (셀 배경 전체에 적용).
```

## 컬러 적용 규칙

| 요소 | 컬러 | 비고 |
|---|---|---|
| S·O 셀 배경 | Primary 강조색 15~20% | 셀 전체 배경 (허용) |
| W·T 셀 배경 | #F3F4F6 연회색 | 셀 전체 배경 (허용) |
| S/O/W/T 레이블 | S·O = 강조색 Bold / W·T = #6B7280 | 텍스트 |
| 불릿 내용 | #1A1A1A Regular | 텍스트 |
| 셀 경계선 | #E5E7EB | 얇은 선 |
