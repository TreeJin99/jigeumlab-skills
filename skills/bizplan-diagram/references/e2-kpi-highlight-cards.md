# [E-2] KPI 하이라이트 카드 레이아웃 레퍼런스

> 발견 문서: F사 IR Deck (P4 Investment Highlight)
> 용도: IR덱 첫 페이지 또는 요약 페이지의 핵심지표 5개 집약

## 구성 요소 상세

### 카드 배치

- 상단 2개 + (중앙 브랜드 원형) + 하단 3개
- 또는 5개 균등 가로 배열 (중앙 원형 없이)

### 각 카드 구성

| 요소 | 형식 |
|---|---|
| 카테고리명 | Regular 소형, #6B7280 (영문 가능) |
| 수치 | Bold 최대형, 강조색 |
| 설명 | Regular 소형, #1A1A1A |
| 카드 외곽 | 흰색 박스 + 회색 구분선 또는 thin border |

### F사 실제 5개 KPI (참고)

| 카드 | 카테고리 | 수치 예시 |
|---|---|---|
| 1 | 시장 규모 | OOO억원 TAM |
| 2 | 성장배율 | OOx 성장 |
| 3 | 트랙션 | OOO개 파트너사 |
| 4 | ARR | OOO만원 |
| 5 | 투자 라운드 | Pre-A |

### 중앙 브랜드 원형 (선택)

수치 카드 중앙에 브랜드 원형 그래픽 또는 로고 배치. 수치들이 중앙 브랜드를 감싸는 형태.

---

## 나노바나나 프롬프트 핵심 지시

```
핵심지표 5개 카드 레이아웃:
- 상단 2개 카드 균등 배치 (좌우)
- 하단 3개 카드 균등 배치

각 카드:
"[카테고리명]" Regular 소형 회색
"[수치]" Pretendard Bold 최대형 강조색
"[설명]" Regular 소형 #1A1A1A
카드 배경: 흰색, 테두리: 얇은 회색 구분선

[선택] 중앙에 브랜드명 또는 로고 원형 배치 — 카드들이 원형을 감싸는 구성
```

## 컬러 적용 규칙

| 요소 | 컬러 | 비고 |
|---|---|---|
| 수치 텍스트 | Primary 강조색 Bold | 핵심 강조 |
| 카테고리명 | #6B7280 Regular | 회색 소형 |
| 설명 텍스트 | #1A1A1A Regular | — |
| 카드 배경 | #FFFFFF | 순수 흰색 |
| 카드 구분선 | #E5E7EB | 얇은 회색 |
| 중앙 원형 배경 | Primary 강조색 (선택) | 도형 (허용) |

## IR덱 vs 사업계획서 사용 기준

- **IR덱**: 첫 페이지 또는 Investment Highlight 섹션에 필수 배치
- **사업계획서**: 기대효과·성과요약 섹션에 활용 가능 (수치 6개면 2×3 변형)
