# [B-4] 막대그래프 + 수치강조패널 레이아웃 레퍼런스

> 발견 문서: G사 IR Deck (P2), C사 (P4), F사 IR (P6·P10)
> 용도: 시장 성장세(CAGR)를 차트와 수치로 이중 표현

---

## 구성 요소 상세

### 좌측 — 막대그래프

| 요소 | 형식 |
|---|---|
| 막대 | 수직 막대 3~5개 (연도별) |
| 미래 막대 | 강조색 (또는 강조색 그라데이션) |
| 현재·과거 막대 | 회색 계열 |
| 막대 위 수치 | Regular 소형 레이블 |
| CAGR 표시 | 막대 위 꺾은선 or 화살표 + "CAGR XX%" Bold |
| X축 | 연도 Regular 소형 |

### 우측 — 수치강조패널

| 요소 | 형식 |
|---|---|
| 수치 | Bold 최대형, 강조색 |
| 단위 | Regular 소형, 수치 옆 또는 위 |
| 설명 | Regular 소형, #6B7280 |
| 패널 외곽 | 흰색 박스 + 얇은 회색 구분선 |

### F사 실제 구성 (참고)

- 막대 4개 (2023~2026), 미래 막대 오렌지 그라데이션
- 우측: "OOO억원", "OO%" CAGR, "OOO만 개" 3개 수치 스택
- CAGR "75x" 배율을 막대 사이 중앙에 강조 배치

## 나노바나나 프롬프트 핵심 지시

```
좌측: 연도별 세로 막대그래프 N개
- 과거·현재 막대: 회색(#9CA3AF)
- 미래·목표 막대: 강조색
- 막대 위: "[연도]" Regular 소형, "[수치]" Regular 소형 레이블
- CAGR: 막대 위 꺾은선 또는 대각 화살표 + "CAGR [X%]" Bold 강조색

우측: 수치 N개 세로 스택
- "[수치①]" Pretendard Bold 최대형 강조색 + "[단위·설명]" Regular 소형 회색 (반복)
- 패널 사이 얇은 회색 구분선

[중요] 막대 색상 값(#색상코드)은 디자인 지시사항. 이미지에 렌더링 금지.
```
