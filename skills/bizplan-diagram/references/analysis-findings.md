## 공통 설득 구조 (전 문서)

### 1. 수치 자체가 도식
수치(11%, 99.2%, 175억)를 카드 중앙에 초대형 Bold 배치. 설명은 아래 소형으로 부속화.
→ A-1, E-2에 적용

### 2. 현재 단계·자사 항목 강조
타임라인·비교표·로드맵에서 현재/자사만 강조색 배경, 나머지 회색·흰색.
→ D-1, D-3, B-2, B-3에 적용

### 3. 풀배너 결론문
슬라이드 하단 전폭 강조색 배경 배너에 1줄 결론 고정.
→ A-3 패턴. 대부분의 문서에서 반복

### 4. 도식+표 병치
포지셔닝맵(좌)+비교표(우), 원형도식(좌)+수치근거표(우) 쌍 배치.
단독 도식보다 직관성+신뢰성 동시 확보.

### 5. 수렴 브라켓
G사·H사에서 전 슬라이드 반복. 좌측 N개 항목 → 우측 결론+CTA.
→ A-4 패턴

## 강조색 사용 원칙 (전 문서 공통)

- **단색 체계**: 1개 강조색을 전 슬라이드 일관 사용 (예외 최소화)
- **텍스트에만**: 수치·브랜드명·핵심어 텍스트에만 강조색. 박스 배경은 흰색
- **예외적 배경색**: 현재단계 박스, 풀배너, 비교표 자사열 — 이 3가지만 배경색 허용
- **도형(아크·원·버블)**: "도형 자체의 색상"이므로 카드 컬러 바 금지 규칙 별개

## IR덱 vs 사업계획서 차이

| 구분 | IR덱 | 정부지원사업 사업계획서 |
|---|---|---|
| 레이아웃 기본단위 | 3컬럼+다단행 (도식 중심) | 2컬럼+표 (텍스트+표 중심) |
| 도식 비중 | 95%+ 슬라이드에 도식 | 50% 슬라이드에 도식, 나머지 표·텍스트 |
| 표 전용 슬라이드 | 거의 없음 | 일정표·자금표·팀구성표 다수 |
| 배경 | 다크 배경(F사) or 흰 배경 | 흰 배경 고정 |
| 수치 강조 강도 | 매우 강함 (수치=도식) | 중간 (수치+설명 병렬) |
