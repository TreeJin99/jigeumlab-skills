## 구성 요소 상세

### 좌측 — 반원 아크 3겹

- 반원(왼쪽 열린 반원 = D자 형태) 아크 3개 겹침
- TAM: 가장 큰 아크, SOM: 가장 작은 아크
- 아크 색상: 진→중→연 (TAM=네이비, SAM=스틸블루, SOM=연회색)
- **주의**: 아크는 "도형"으로 지시 — 카드 컬러 바 금지 규칙과 별개 (도형은 허용)

### 우측 — 3행 구분선 패널

각 행 구성:
| 요소 | 형식 | 크기 |
|---|---|---|
| 약어 (TAM / SAM / SOM) | Bold | 대형 |
| 한글 명칭 (전체시장 / 유효시장 / 목표시장) | Regular | 소형 |
| 시장 규모 수치 | Bold | 강조색 |
| 정의 1줄 + 출처 | Regular | 소형 회색 |

행 사이: 얇은 구분선 (1px, 회색)

## 나노바나나 프롬프트 핵심 지시

```
좌측: 반원 형태의 도형(D자형) 3겹 — TAM(가장 큰 아크, 네이비), SAM(중간 아크, 스틸블루), SOM(가장 작은 아크, 연회색)
우측: 3행 정보 패널, 행 사이 얇은 회색 구분선
- 행1: "TAM" (Bold 대형, 네이비) + "전체시장" (Regular 소형) + "OOO" (Bold 강조색) + "정의 (출처)"
- 행2: "SAM" (Bold 대형, 스틸블루) + "유효시장" (Regular 소형) + "OOO" (Bold 강조색) + "정의 (출처)"
- 행3: "SOM" (Bold 대형, 연회색 계열) + "목표시장" (Regular 소형) + "OOO" (Bold 강조색) + "정의 (출처)"
```
