# [A-4] 수렴 브라켓 레이아웃 레퍼런스

> 발견 문서: G사 IR Deck (전 슬라이드 반복), H사 Deck v2
> 용도: 다수 항목을 나열한 뒤 하나의 결론으로 수렴시키는 설득 구조

---

## 레이아웃 구조

```
┌─────────────────────────────────────────────────────┐
│                                                      │
│  ┌─────────────────┐                                │
│  │ [01] 항목 제목   │─────┐                         │
│  │ • 세부내용       │     │                         │
│  └─────────────────┘     │      ┌──────────────┐   │
│                            ├────>│  결론 Bold   │   │
│  ┌─────────────────┐     │      │  1~2줄       │   │
│  │ [02] 항목 제목   │─────┤      ├──────────────┤   │
│  │ • 세부내용       │     │      │ [CTA 배너]   │   │
│  └─────────────────┘     │      │ 강조색 배경  │   │
│                            │      └──────────────┘   │
│  ┌─────────────────┐     │                         │
│  │ [03] 항목 제목   │─────┘                         │
│  │ • 세부내용       │                               │
│  └─────────────────┘                               │
│                                                      │
└─────────────────────────────────────────────────────┘
```

## 구성 요소 상세

### 좌측 — 항목 나열 (2~4개)

| 요소 | 형식 |
|---|---|
| 번호 태그 | [01] [02] [03] — 강조색 배경 소형 박스 |
| 제목 | SemiBold 중형 |
| 세부내용 | 불릿(•) Regular 소형 2~3줄 |
| 카드 외곽 | 흰색 박스 + 얇은 회색 테두리 |

### 중앙 — 수렴 선

- 각 항목 우측 끝에서 꺾임 선(브라켓) 연결
- 선 색상: 강조색 또는 회색 얇은 선
- 화살표 없이 선만 (또는 얇은 chevron)

### 우측 — 결론 + CTA

| 요소 | 형식 |
|---|---|
| 결론 텍스트 | Bold 중형~대형, 1~2줄 |
| CTA 박스 | 강조색 배경 + 흰색 Bold 텍스트 1줄 |

---

## 나노바나나 프롬프트 핵심 지시

```
좌측에 [번호 태그 강조색 배경]이 붙은 카드 N개를 세로 나열.
각 카드에서 우측으로 꺾이는 얇은 선이 수렴하여 우측 결론 텍스트로 연결.
우측: "[결론 텍스트]" (Bold 중형) + 아래 "[CTA 문장]" (강조색 배경 박스, 흰색 텍스트 Bold)

브라켓 선 = 얇은 회색 또는 강조색 직선 + 오른쪽 꺾임. 두꺼운 화살표 삼각형 금지.
```

## 실제 사용 예시 (G사)

- 경쟁우위 3항목 → "G사만의 차별화" 결론 수렴
- 핵심역량 3항목 → "시장 선점 가능" CTA 배너
- 준비현황 3항목 → "즉시 사업화 가능" 결론

## 컬러 적용 규칙

| 요소 | 컬러 | 비고 |
|---|---|---|
| 번호 태그 배경 | Primary 강조색 | 소형 박스 배경 (허용) |
| CTA 박스 배경 | Primary 강조색 | 배너 배경 (허용) |
| 수렴 선 | 강조색 또는 #6B7280 | 얇은 선 |
| 카드 외곽 | #E5E7EB (연회색) | 테두리만 |
| 결론 텍스트 | #1A1A1A Bold | 배경 없음 |
| CTA 텍스트 | #FFFFFF Bold | 배너 안 |
