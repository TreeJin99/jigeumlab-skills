# [B-3] 포지셔닝 매트릭스 2×2 레이아웃 레퍼런스

> 발견 문서: A사 KOICA 사업계획서 (P5), C사 Appendix (P17)
> 용도: 경쟁사 대비 자사 포지션을 2개 축으로 시각화

---

## 레이아웃 구조

```
┌─────────────────────────────────────────────────────┐
│                    [Y축 상단 레이블]                 │
│                           ↑                          │
│              2사분면      │      1사분면(자사)       │
│           ○ 경쟁사B      │    ★ [자사명]            │
│                           │    (강조색 버블)         │
│  [X축 좌측] ─────────────┼───────────── [X축 우측] │
│              3사분면      │      4사분면             │
│           ○ 경쟁사C      │    ○ 경쟁사D             │
│                           ↓                          │
│                    [Y축 하단 레이블]                 │
└─────────────────────────────────────────────────────┘
```

---

## 구성 요소 상세

### 축 설정

| 요소 | 형식 |
|---|---|
| X축 | 가로 직선 — 좌끝·우끝에 상반되는 레이블 |
| Y축 | 세로 직선 — 상끝·하끝에 상반되는 레이블 |
| 레이블 | Regular 소형, #6B7280 |
| 교차점 | 중앙 십자 교차 (원점 표시 없음) |

### 버블 (경쟁사 / 자사)

| 요소 | 자사 | 경쟁사 |
|---|---|---|
| 도형 | 원형 버블 (중간~대형) | 원형 버블 (소형) |
| 배경색 | Primary 강조색 | #D1D5DB (회색) |
| 텍스트 | 브랜드명 Bold 흰색 | 경쟁사명 Regular 회색 |

### C사 실제 축 예시 (참고)

- X축: "글로벌 범용성" ↔ "K푸드 정체성"
- Y축: "건강 낮음" ↔ "건강 높음"
- 자사 위치: 우상단 1사분면 (K푸드 + 고건강)

---

## 나노바나나 프롬프트 핵심 지시

```
중앙에 X축(가로)과 Y축(세로) 십자 교차선. 선 색상 #1A1A1A 얇은 직선.
X축 좌끝: "[레이블A]" / 우끝: "[레이블B]" (Regular 소형 회색)
Y축 상끝: "[레이블C]" / 하끝: "[레이블D]" (Regular 소형 회색)

버블 배치:
- "[자사명]" — [1/2/3/4]사분면에 중간 크기 원형 버블, 강조색 배경, 흰색 Bold 텍스트
- "[경쟁사명]" — 해당 사분면에 소형 원형 버블, 회색 배경, 회색 Regular 텍스트 (반복)

버블 = "원형 도형" 지시. 카드 컬러 바 규칙과 별개. 두꺼운 테두리 금지.
```

## 컬러 적용 규칙

| 요소 | 컬러 | 비고 |
|---|---|---|
| 자사 버블 배경 | Primary 강조색 | 도형 색상 (허용) |
| 경쟁사 버블 배경 | #D1D5DB | 회색 도형 |
| 축선 | #1A1A1A | 얇은 직선 |
| 레이블 텍스트 | #6B7280 | Regular 소형 |
| 배경 | #FFFFFF | 순수 흰색 |
