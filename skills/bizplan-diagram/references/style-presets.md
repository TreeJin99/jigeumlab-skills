# 나노바나나 스타일 프리셋 (제출용 · 발표용)

> 도식 용도에 따라 둘 중 하나를 프롬프트에 포함. Phase 1에서 용도 확인.
> - **제출용**: HWP 사업계획서·정부 제출본 → 미니멀·순백·신뢰감 (기본값)
> - **발표용**: IR덱·피칭·발표 슬라이드 → 그라데이션·아이콘·입체 임팩트

두 프리셋 모두 공통: Pretendard, 한글 오타 없이 정확, 16:9, 수치 Bold 최대형.

---

## 프리셋 A — 제출용 (미니멀 / 정부 제출 톤)

정부 사업계획서 HWP에 삽입되는 도식. 절제·신뢰감 우선.

```
## 스타일 지시 (※ 아래 값은 이미지에 렌더링 금지)
- 폰트: Pretendard (Bold/SemiBold/Regular 계층 구분)
- 배경: 순수한 흰색(#FFFFFF) 단색 (텍스처·그라데이션 금지)
- 기본 텍스트: #1A1A1A
- 강조 색상: [해당 텍스트 요소]에만 적용. 카드/박스/헤더/외곽 영역 사용 금지
- 분위기: 한국 공문서·사업계획서 톤, 깔끔, 미니멀, 신뢰감

[중요] 폰트명·색상 코드는 디자인 지시사항이며 이미지에 렌더링할 텍스트가 아닙니다.

[금지 요소]
카드 상단·하단·좌우에 컬러 바·라인·띠지·헤더 영역 금지.
카드 = 흰색 박스 + 매우 가벼운 drop shadow + 내부 텍스트만.
화살표 = 얇은 선 + chevron(>) 형태만. 두꺼운 삼각형 금지.
Step 번호 원형 배지·다이아몬드 bullet 금지.
Heavy drop shadow·glow·네온 효과 금지.
빈 공간의 장식 컬러 도형 금지.
```

---

## 프리셋 B — 발표용 (프리미엄 / IR·피칭 톤)

발표 슬라이드·IR덱용. 시각 임팩트로 핵심 메시지를 10초에 꽂음.
*검증: PlanDoc 차별점 벤다이어그램 v3 — 그라데이션·아이콘·중앙 글로우 조합이 미니멀 대비 명확히 우위.*

```
## 스타일 지시 (※ 아래 값은 이미지에 렌더링 금지)
- 폰트: Pretendard (Bold/SemiBold/Regular 계층 구분), 한글 오타 없이 정확히
- 배경: 흰색~아주 옅은 회색, 여백 넉넉, 모던 테크 스타트업 발표자료 톤
- 기본 텍스트: #1A1A1A
- 핵심 도형: 부드러운 그라데이션(강조색 계열) + 은은한 입체 soft shadow
- 강조/자사 요소: 중앙 또는 핵심 영역에 은은한 glow로 시선 집중. 골드 포인트 1곳
- 아이콘: 각 영역 의미에 맞는 깔끔한 미니 아이콘 1개씩 (과하지 않게)
- 분위기: 세련, 신뢰감, 고급스러운 프레젠테이션 품질

[중요] 폰트명·색상 코드는 디자인 지시사항이며 이미지에 렌더링할 텍스트가 아닙니다.

[허용 요소 — 발표용 한정]
도형 그라데이션, 부드러운 입체 shadow, 핵심 1곳 glow, 영역별 미니 아이콘, glassmorphism 카드.

[금지 요소]
난잡한 네온·무지개색·과도한 장식. 빈 공간의 의미 없는 컬러 도형.
가독성 해치는 진한 배경. 한글 깨짐·오타.
```
