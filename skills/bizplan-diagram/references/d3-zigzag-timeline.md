# [D-3] 마일스톤 지그재그 타임라인 레이아웃 레퍼런스

> 발견 문서: C사 IR Deck (P11), F사 IR (P5·P18·P27)
> 용도: 연도/분기별 마일스톤을 상하 교번 배치로 공간 효율 극대화

---

## 레이아웃 구조

```
┌──────────────────────────────────────────────────────────┐
│                                                           │
│  [아이콘+제목]              [아이콘+제목]                │  ← 홀수: 위
│       │                          │                        │
│  ─────●─────────────────────────●──────────────────── ▶ │  ← 중앙 타임라인 선
│       │              │           │              │         │
│              [아이콘+제목]              [아이콘+제목]     │  ← 짝수: 아래
│                                                           │
│  2026 Q1          2026 Q3         2027 Q1         2027 Q3 │  ← 날짜 레이블
│  (현재: 강조색)                                           │
│                                                           │
└──────────────────────────────────────────────────────────┘
```

---

## 구성 요소 상세

### 중앙 타임라인 선

| 요소 | 형식 |
|---|---|
| 선 | 가로 직선 (얇은 #9CA3AF 또는 강조색) |
| 마일스톤 도트 | 원형 도트 — 현재: 강조색 채움 / 과거: 회색 채움 / 미래: 빈 원 |
| 화살표 | 선 우끝에 chevron(>) 방향 표시 |

### 상하 교번 배치

- **홀수 번째** (Mil.1·3·5): 도트에서 **위쪽**으로 수직선 + 아이콘+제목+설명
- **짝수 번째** (Mil.2·4·6): 도트에서 **아래쪽**으로 수직선 + 아이콘+제목+설명

### 각 마일스톤 구성

| 요소 | 형식 |
|---|---|
| 아이콘 | 선택적 (있으면 텍스트 위) |
| 날짜/분기 | Regular 소형, 도트 아래 또는 위 선상 |
| 제목 | SemiBold 중형 |
| 설명 | Regular 소형, 1~2줄 |
| 현재 도트 | 강조색 채움 원 + 해당 텍스트 강조색 Bold |

### F사 시제 색상 코드 (참고)

| 시제 | 도트 색상 | 텍스트 색상 |
|---|---|---|
| 과거 | #6EE7B7 (민트 채움) | #6B7280 |
| 현재 | Primary 강조색 채움 | Primary 강조색 Bold |
| 미래 | 빈 원 (테두리만) | #9CA3AF |

---

## 나노바나나 프롬프트 핵심 지시

```
가로 중앙선 + 원형 도트 N개로 구성된 타임라인:
- 도트 색상: 과거=회색 채움 원, 현재=강조색 채움 원(크게), 미래=빈 원(테두리만)
- 홀수 마일스톤: 도트 위쪽에 수직선 + "[날짜]" 소형 + "[제목]" SemiBold + "[설명]" 소형
- 짝수 마일스톤: 도트 아래쪽에 수직선 + 동일 구성
- 타임라인 선 우끝: 화살표(>) 방향 표시

날짜 레이블: 타임라인 선 아래 또는 각 도트 근처에 Regular 소형

[중요] 색상 코드는 디자인 지시사항. 이미지에 렌더링 금지.
도트 = "원형 도형". 카드 컬러 바 금지 규칙과 별개.
```
