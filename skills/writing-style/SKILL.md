# 사업계획서 문체·서술 규칙

평가위원이 **10초 안에 핵심 가치를 파악**하고 **80점 이상**을 부여할 수 있게 작성.

## Workflow

### Phase 1 — 문체 교정 (종결형)
- `~합니다/~입니다/~습니다` → `~함/~임/~됨` 전환
- 수동태 → 능동태: "~가 개선됨" → "~를 개선함"
- 한 문장당 절 2개 초과 시 분리

### Phase 2 — 정량화 검사
- 추상 표현 목록 추출 후 수치로 대체
- 미검증 수치: `*(확인 필요)*` 마커 부착
- 클라이언트 미제공 수치 임의 생성 절대 금지

### Phase 3 — 금지어·수식어 제거
- 절대 금지어 스캔 → Constraints §금지어 목록 기준
- 빈 수식어 → 수치 기반 표현으로 대체

### Phase 4 — 구조·표기 통일
- 개조식 계층 (ㅇ → - → •) 정렬
- 단위·숫자 표기 Constraints §숫자 규칙 준수
- 도식 자리표시자: `[도식 위치] — {유형} ({내용 요약})`

## Constraints

### 문체
- 종결형 허용: `~함, ~임, ~됨, ~것임, ~한다, ~이다, ~했음`
- 절대 금지: `~합니다, ~입니다, ~습니다, ~해요, ~이에요, ~네요, ~죠`
- 문장 구조: 절 2개 이하, 능동태 우선, 수식어 최소화

### 개조식 계층
```
ㅇ 중항목 (섹션 내 주요 논점)
  - 세부 항목 (근거·예시)
  • 강조 항목 (핵심 수치 또는 결론)
  ✓ 완료/보유 사실
  → 결론 도출
```

### 정량화 기준
- 섹션당 5개 이상, 전체 30개 이상 수치 포함
- 출처: `(한국무역협회, 2024)` 형식
- 미검증: `(추정)` 또는 `(가상)` 태그 필수

### 금지어
절대 금지: 혁신적, 혁명적, 최초, 유일한, 최고, 완벽한, 무한한, 패러다임, 게임체인저, 대변혁, 파괴적 혁신

대체 표현: "기존 대비 85% 개선된 솔루션", "국내 유일 특허 기반 접근법 (특허 4건)"

### 숫자·단위 표기
- 만 단위: `2,320만명` (23.2M 아님)
- 퍼센트: `71%` (단위 공백 없음)
- 금액: `4,000만원`, 1억 미만 백만원 / 1억 이상 억원
- 외래어: 한글 우선, 업계 표준(B2C·CAGR·SaaS)은 영문 허용

### 강조 패턴
- **굵게**: 핵심 수치, 결론 문장
- `[카테고리]`: 항목 분류
- `→` 결론 도출
- `*(확인 필요)*`: 미확정 항목

### 문서 구조
- 헤딩 최대 3단계 (#/##/###), `####` 이하 금지
- 번호 구분자: `.` (1.1. 형식)
- 섹션 제목 볼드 처리 금지

## Output Schema

```
[나쁜 예] 혁신적인 AI 솔루션입니다. 많은 고객이 사용 중입니다.
[좋은 예] AI 기반 처리시간 97% 단축 솔루션임. 누적 12,000명 사용, YoY 333% 성장함.

[나쁜 예] 빠르게 성장하고 있습니다.
[좋은 예] 매출 3년간 연평균 200% 성장함.
```

## Anti-Triggers

- **영어 문서 작성**: 영어 보고서·이메일 — 한국어 사업계획서 문체 규칙 비적용
- **일반 문서 편집**: SNS·이메일·일반 보고서 — 사업계획서 외 문서는 별도 문체 기준
- **공고문 분석**: "이 공고 요약해줘" → notice-quickread 영역
- **사업계획서 초안 작성 (콘텐츠)**: 문체 규칙 적용은 본 스킬, 내용 작성은 biz-plan 스킬
